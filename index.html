3R open chain linkage freefall using forward Dynamics

The project successfully demonstrated the application of forward dynamics using the Lagrangian formulation to simulate the free fall of a 3R planar arm. Strawberry Stacker

Theme Description: Reduce physical toil on strawberry farms by making strawberry stacking for transport an automated process using multicopters. Concepts of control systems, image processing, mission planners and distributed systems, while using the typical software stacks used to control flying vehicles and plan missions for them. The final goal is to build an application where we control a team of multicopters and achieve the task of picking strawberry boxes from a field and stacking them onto a transport trailer. These tasks take place in a simulated environment that allows us to prototype our application quickly and smoothly, while ensuring application transferability to real hardware.
